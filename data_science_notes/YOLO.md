# YOLO Object Detection algorithm

YOLO = You Only Look Once

State of the art algorithm for image localisation, or object detection, introduced in 2015.

## Image Localisation

The preocess of identifying the current location of one or multiple objects using bounding boxes, which correspond to rectangular shaes around objects.

Not to be confused with image classification or image recognition.
