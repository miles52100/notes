# conda

An open source package management system.
Documentation [here](https://docs.conda.io/en/latest/). For any language - Python, R, Ruby, Lua etc.

It's included in all versions of Anaconda and Miniconda

  * default config: install and maage over 7500 packages at `repo.anaconda`. These are built, reviewed and maintained by Anaconda.

  * Can be combined with CI systems such as GitHub Actions, to provide frequent, automated testing of code

  * conda-forge a community package channel 


## Environments
Seem to be stored, for macOS, in `/opt/anaconda3/envs`
The subdirectory `share/info` contains all metadata about a package.
