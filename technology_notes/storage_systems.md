## Types of data storages

### data lake

- heterogeneous
- stored in raw form, usually blobs
- usually a single store of data

### data hub

- homogenizes data
- serves data in possibly multiple desired formats
- adds value, dedupes, quality, security
- standardizes set of query services
- not limited to operational data
- data usually unintegrated (c.f. data warehouse) and at different grains

### data warehouse

- aka enterprise data warehouse (EDW)
- system used for reporting and data analysis
- core component of biz intel.
- central repos of data, integrated from disparate sources
- current and historical data
- two main workflows for building a data warehouse: ETL (extract transform load) and ELT
